PROYECTO FINAL
____________________________

El principal objetivo de este proyecto es el desarrollo de un videojuego en tercera persona, en el que el jugador debe ir esquivando los zombis del lugar, recogiendo elementos para activar bombas temporizadas y así pasar de nivel.
 


!!AVISO!!
Se necesita de un teclado y un mouse para jugar.
